Zainstalować XAMPP potrzebny do obsługi lokalnego serwera aplikacji i bazy danych.
Po zainstalowaniu podmienić folder ~\xampp\htdocs na folder htdocs zamieszczony w repozytorium.
Uruchomić XAMPP Control Panel i włączyć moduły Apache oraz MySQL.
Włączyć przeglądarkę i wpisać adres localhost/phpmyadmin.
Stworzyć bazę danych o nazwie "serwis".
Zaimportować do bazy danych plik serwis.sql.
Wpisać adres localhost.

Do implementacji prototypu wykorzystane zostały Baza danych, PHP, HTML oraz CSS.
